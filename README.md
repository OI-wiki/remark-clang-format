# remark-clang-format

## Install

```bash
npm install remark-clang-format
```

## Usage

add `"remark-clang-format"` to `"plugins"` in `.remarkrc`

## 发布

`yarn release` 选择合适的版本号，推到 GitHub 上后 GitHub Actions 自动发布到 GitHub Releases 和 npm。
